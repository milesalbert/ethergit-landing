Proof of Contribution

We're rethinking how developers contribute to projects and how they're rewarded.

Incentivized Development

We are creating a platform for open source software that incentivizes contributions with ownership of the project.

Stakeholder Voting

Developers will be incentivized to contribute by receiving tokens that can be used to vote on future changes.

Completely Decentralized

Since EtherGit is built using the Ethereum blockchain, virtually any governance scheme can be implemented over repositories.

Full Abstraction

Every repository has a .nomic file, where the creator must first define the rules for rulemaking before evolving the governance scheme.
